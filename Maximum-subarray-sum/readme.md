# Maximum Subarray Sum

## Problem Statement

Given an array of `N` integers (which may include negative numbers), find the maximum possible sum of any contiguous subarray.

### Input

* The first line contains an integer `N`.
* The second line contains `N` space-separated integers.

### Output

Print the maximum subarray sum.

### Constraints

* `1 ≤ N ≤ 10^5`
* `-10^9 ≤ Array[i] ≤ 10^9`

### Example

**Input**

```text
8
-2 -3 4 -1 -2 1 5 -3
```

**Output**

```text
7
```

### Explanation

The contiguous subarray `4 -1 -2 1 5` has the maximum sum of `7`.
