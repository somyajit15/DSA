# Find the Missing Number

## Problem Statement

Given an array containing `N-1` distinct integers from `1` to `N`, find the missing number.

The array contains all numbers from `1` to `N` except one. Your task is to identify and print the missing number.

### Input

* The first line contains an integer `N`.
* The second line contains `N-1` space-separated integers.

### Output

Print the missing number.

### Constraints

* `2 ≤ N ≤ 10^5`
* The array contains distinct integers from `1` to `N`.
* Exactly one number is missing.

### Example

**Input**

```text
5
1 2 3 5
```

**Output**

```text
4
```

### Explanation

The numbers should be `1, 2, 3, 4, 5`, but `4` is missing from the array, so the output is `4`.
