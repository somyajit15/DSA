# Count Even and Odd Numbers

## Problem Statement

Given an array of `N` integers, count the number of even and odd elements in the array.

### Input

* The first line contains an integer `N`.
* The second line contains `N` space-separated integers.

### Output

Print the count of even numbers followed by the count of odd numbers.

### Constraints

* `1 ≤ N ≤ 10^5`
* `-10^9 ≤ Array[i] ≤ 10^9`

### Example

**Input**

```text
6
2 5 8 7 10 3
```

**Output**

```text
Even: 3
Odd: 3
```

### Explanation

The even numbers are `2`, `8`, and `10`, while the odd numbers are `5`, `7`, and `3`. Therefore, the array contains 3 even numbers and 3 odd numbers.
