# Reverse an Array

## Problem Statement

Given an array of `N` integers, reverse the order of its elements and print the resulting array.

### Input

* The first line contains an integer `N`.
* The second line contains `N` space-separated integers.

### Output

Print the elements of the array in reverse order.

### Constraints

* `1 ≤ N ≤ 10^5`
* `-10^9 ≤ Array[i] ≤ 10^9`

### Example

**Input**

```text
5
10 20 30 40 50
```

**Output**

```text
50 40 30 20 10
```

### Explanation

The elements of the array are printed in the opposite order of their original arrangement.
