# Find the Second Largest Element

## Problem Statement

Given an array of `N` integers, find and print the second largest distinct element in the array.

### Input

* The first line contains an integer `N`, the number of elements in the array.
* The second line contains `N` space-separated integers.

### Output

Print the second largest distinct element in the array.

### Constraints

* `2 ≤ N ≤ 10^5`
* `-10^9 ≤ Array[i] ≤ 10^9`

### Example

**Input**

```text
5
10 20 5 15 8
```

**Output**

```text
15
```

### Explanation

The largest element in the array is `20`, and the second largest distinct element is `15`.
