# Find the Majority Element

## Problem Statement

Given an array of `N` integers, find the element that appears more than `N/2` times in the array.

You may assume that a majority element always exists.

### Input

* The first line contains an integer `N`.
* The second line contains `N` space-separated integers.

### Output

Print the majority element.

### Constraints

* `1 ≤ N ≤ 10^5`
* `-10^9 ≤ Array[i] ≤ 10^9`

### Example

**Input**

```text
7
2 2 1 1 2 2 2
```

**Output**

```text
2
```

### Explanation

The element `2` appears 5 times in the array, which is more than `N/2` (3.5). Therefore, `2` is the majority element.
