# Count the Number of Nodes in a Singly Linked List

## Problem Statement

Given a singly linked list, count and print the total number of nodes present in the list.

### Input

* The first line contains an integer `N`, representing the number of nodes.
* The second line contains `N` space-separated integers representing the node values.

### Output

Print the total number of nodes in the linked list.

### Constraints

* `1 ≤ N ≤ 10^5`
* `-10^9 ≤ Node Value ≤ 10^9`

### Example

**Input**

```text
5
10 20 30 40 50
```

**Output**

```text
5
```

### Explanation

The linked list contains five nodes: `10 → 20 → 30 → 40 → 50`. Therefore, the total number of nodes is `5`.
