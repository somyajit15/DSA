# Rotate an Array by One Position

## Problem Statement

Given an array of `N` integers, rotate the array to the right by one position.

### Input

* The first line contains an integer `N`.
* The second line contains `N` space-separated integers.

### Output

Print the array after rotating it to the right by one position.

### Constraints

* `1 ≤ N ≤ 10^5`
* `-10^9 ≤ Array[i] ≤ 10^9`

### Example

**Input**

```text
5
1 2 3 4 5
```

**Output**

```text
5 1 2 3 4
```

### Explanation

The last element moves to the first position, and all other elements shift one position to the right.
