# Find the Frequency of a Given Element

## Problem Statement

Given an array of `N` integers and an integer `X`, count how many times `X` appears in the array.

### Input

* The first line contains an integer `N`.
* The second line contains `N` space-separated integers.
* The third line contains an integer `X`.

### Output

Print the frequency of `X` in the array.

### Constraints

* `1 ≤ N ≤ 10^5`
* `-10^9 ≤ Array[i], X ≤ 10^9`

### Example

**Input**

```text
7
2 5 2 8 2 3 5
2
```

**Output**

```text
3
```

### Explanation

The element `2` appears 3 times in the array, so the output is `3`.
