# Remove Duplicates from a Sorted Array

## Problem Statement

Given a sorted array of integers, remove all duplicate elements and print the unique elements in their original order.

### Input

* The first line contains an integer `N`, the number of elements in the array.
* The second line contains `N` space-separated integers in sorted order.

### Output

Print the array after removing all duplicate elements.

### Constraints

* `1 ≤ N ≤ 10^5`
* `-10^9 ≤ Array[i] ≤ 10^9`
* The array is sorted in non-decreasing order.

### Example

**Input**

```text
8
1 1 2 2 3 4 4 5
```

**Output**

```text
1 2 3 4 5
```

### Explanation

The array contains duplicate values `1`, `2`, and `4`. After removing the duplicates, the remaining unique elements are `1 2 3 4 5`.
