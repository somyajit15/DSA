# Find the Pair with the Given Sum

## Problem Statement

Given an array of `N` integers and an integer `K`, determine whether there exists a pair of elements whose sum is equal to `K`.

If such a pair exists, print `Yes`; otherwise, print `No`.

### Input

* The first line contains an integer `N`.
* The second line contains `N` space-separated integers.
* The third line contains an integer `K`.

### Output

Print `Yes` if there exists a pair whose sum is equal to `K`; otherwise, print `No`.

### Constraints

* `2 ≤ N ≤ 10^5`
* `-10^9 ≤ Array[i], K ≤ 10^9`

### Example

**Input**

```text
6
2 7 11 15 3 6
9
```

**Output**

```text
Yes
```

### Explanation

The pair `(2, 7)` has a sum of `9`, so the output is `Yes`.
