# Find the Maximum Product of Two Elements

## Problem Statement

Given an array of `N` integers, find the maximum product that can be obtained by multiplying any two distinct elements of the array.

### Input

* The first line contains an integer `N`.
* The second line contains `N` space-separated integers.

### Output

Print the maximum product of any two distinct elements.

### Constraints

* `2 ≤ N ≤ 10^5`
* `-10^9 ≤ Array[i] ≤ 10^9`

### Example

**Input**

```text
6
2 7 1 8 5 9
```

**Output**

```text
72
```

### Explanation

The two largest elements are `9` and `8`. Their product is `9 × 8 = 72`, which is the maximum possible product.
