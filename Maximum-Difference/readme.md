# Find the Maximum Difference

## Problem Statement

Given an array of `N` integers, find the maximum difference between any two elements such that the larger element appears after the smaller element.

### Input

* The first line contains an integer `N`.
* The second line contains `N` space-separated integers.

### Output

Print the maximum difference.

### Constraints

* `2 ≤ N ≤ 10^5`
* `-10^9 ≤ Array[i] ≤ 10^9`

### Example

**Input**

```text
6
7 1 5 3 6 4
```

**Output**

```text
5
```

### Explanation

The maximum difference is obtained by subtracting `1` from `6`, resulting in `5`.
