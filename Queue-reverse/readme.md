# Reverse a Queue

## Problem Statement

Given a queue containing `N` integers, reverse the order of its elements.

Print the queue after reversing it.

### Input

* The first line contains an integer `N`, the number of elements in the queue.
* The second line contains `N` space-separated integers.

### Output

Print the elements of the reversed queue.

### Constraints

* `1 ≤ N ≤ 10^5`
* `-10^9 ≤ Queue[i] ≤ 10^9`

### Example

**Input**

```text
5
10 20 30 40 50
```

**Output**

```text
50 40 30 20 10
```

### Explanation

The original queue is:

`Front → 10 20 30 40 50`

After reversing:

`Front → 50 40 30 20 10`
