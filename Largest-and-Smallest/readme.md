# Find the Largest and Smallest Element

## Problem Statement

Given an array of `N` integers, find and print the largest and the smallest element present in the array.

### Input

* The first line contains an integer `N`.
* The second line contains `N` space-separated integers.

### Output

Print the largest element followed by the smallest element.

### Constraints

* `1 ≤ N ≤ 10^5`
* `-10^9 ≤ Array[i] ≤ 10^9`

### Example

**Input**

```text
6
12 5 8 20 3 15
```

**Output**

```text
Largest: 20
Smallest: 3
```

### Explanation

The largest value in the array is `20`, while the smallest value is `3`.
