# Check if a Linked List is a Palindrome

## Problem Statement

Given the head of a singly linked list, determine whether the linked list is a palindrome.

A linked list is called a palindrome if it reads the same forward and backward.

Print **"YES"** if it is a palindrome; otherwise, print **"NO"**.

### Input

* The first line contains an integer `N`.
* The second line contains `N` space-separated integers representing the linked list.

### Output

Print `YES` if the linked list is a palindrome; otherwise, print `NO`.

### Constraints

* `1 ≤ N ≤ 10^5`
* `-10^9 ≤ Node Value ≤ 10^9`

### Example

**Input**

```text
5
1 2 3 2 1
```

**Output**

```text
YES
```

### Explanation

The linked list reads the same from left to right and right to left, so it is a palindrome.
