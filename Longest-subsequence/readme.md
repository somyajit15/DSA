# Find the Longest Consecutive Sequence

## Problem Statement

Given an unsorted array of `N` integers, find the length of the longest sequence of consecutive integers.

The consecutive elements do not need to appear next to each other in the array.

### Input

* The first line contains an integer `N`.
* The second line contains `N` space-separated integers.

### Output

Print the length of the longest consecutive sequence.

### Constraints

* `1 ≤ N ≤ 10^5`
* `-10^9 ≤ Array[i] ≤ 10^9`

### Example

**Input**

```text
6
100 4 200 1 3 2
```

**Output**

```text
4
```

### Explanation

The longest consecutive sequence is `1, 2, 3, 4`, which has a length of `4`.
